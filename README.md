# Work Log

```bash
############################################################
######         ********* Work Log **********          ######
############################################################
##  - Manage your daily todos in a nice markdown format
##  - Generate a report for [slack|stand]ups
############################################################
## Options
##----------------------------------------------------------
##     -e | --edit
##        Edit your worklog
##     -E | --edit-script
##        Edit this script
##     -y | --yesterday
##        Edit your worklog
##     -dS | --date-start
##        Add new todo item
##     -a | --add
##        Add new todo item
##     -s | --snippet
##        Add a new snippet
##     -r | --report
##        generate a report for the last 2 days
##     -S | --summary
##        detailed summary
##     -D | --done <number>
##        Mark a todo item as done by number
##     -u | --undone <number>
##        Mark a todo item as undone by number
##     -U | --upgrade
##        Upgrade wlog files. backup recommended
##
##     -h | --help
##        Display this help message
##
##----------------------------------------------------------
## Environment Variables
##----------------------------------------------------------
##
## WLOG_HOME
##   Directory for all the wlogs.
##   defaults to $HOME/wlog
##
############################################################
```
